## 🛠️ Tech Stack

This portfolio is built using modern web technologies:

* Next.js 14
* TypeScript
* Tailwind CSS
* Framer Motion

## ⚡ Getting Started

### Clone the Repository

```bash
git clone <repository-url>
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

in your browser.

## 📦 Production Build

```bash
npm run build
npm start
```
